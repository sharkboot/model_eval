# AlpacaEval 数据集分析报告

---

## 1. 简介

### 1.1 来源

AlpacaEval是由Stanford发布的指令跟随评测基准，于2024年4月发布长度控制版本，论文发表于arXiv（arXiv:2404.04475）。该基准包含805道指令。

- **发布机构**：Stanford
- **发布时间**：2024年4月
- **论文链接**：https://arxiv.org/abs/2404.04475
- **数据集链接**：待补充
- **项目仓库**：https://github.com/tatsu-lab/alpaca_eval

### 1.2 目标

AlpacaEval旨在解决评估大语言模型的指令跟随能力的问题。

- 主要目标：评估大语言模型的指令跟随能力
- 解决问题：
  - 自动评估存在长度偏差
  - 缺乏标准化的指令跟随评测

### 1.3 应用场景

AlpacaEval的应用场景涵盖了从模型评估到学术研究的多个层面。该数据集不仅能够用于评估指令执行质量方面的能力表现，还可以作为模型对比的标准化基准。
