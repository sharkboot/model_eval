# AlignBench 数据集分析报告

---

## 1. 简介

### 1.1 来源

AlignBench是由清华大学发布的中文对齐评测基准，于2023年11月正式发布，论文发表于arXiv（arXiv:2311.18743）。该基准从多个维度评估大语言模型在中文语境下的对齐质量。

- **发布机构**：清华大学
- **发布时间**：2023年11月
- **论文链接**：https://arxiv.org/abs/2311.18743
- **数据集链接**：待补充
- **项目仓库**：https://github.com/THUDM/AlignBench

### 1.2 目标

AlignBench旨在解决评估大语言模型在中文语境下的对齐质量的问题。

- 主要目标：评估大语言模型在中文语境下的对齐质量
- 解决问题：
  - 缺乏中文对齐评测基准
  - 现有对齐评测多为英文

### 1.3 应用场景

AlignBench的应用场景涵盖了从模型评估到学术研究的多个层面。该数据集不仅能够用于评估中文对齐质量方面的能力表现，还可以作为模型对比的标准化基准。
